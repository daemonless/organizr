## Deployment

### Podman Compose

```yaml
services:
  organizr:
    image: ghcr.io/daemonless/organizr:latest
    container_name: organizr
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - /path/to/containers/organizr:/config
    ports:
      - 8083:8083
    restart: unless-stopped
```

### Podman CLI

```bash
podman run -d --name organizr \
  -p 8083:8083 \
  -e PUID=@PUID@ \
  -e PGID=@PGID@ \
  -e TZ=@TZ@ \
```
